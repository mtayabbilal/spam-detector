About Application

The major speciality of this application is that, it is purely developed by us. No logic or any other part is copied. We are feeling proud that we have learned something new and special, when many of our fellows are trying to impress the teachers..!!


The spam detector is an application, that sets a specific criteria of keywords, directly from stored data in the database or it takes from user at the run time. Then the user play it and put a lengthy textual content and begins the test. The application process the given data according to specified criteria and finally present the result. In the result we get number of words captured during test, a list of found words, and processed paragraph.


Different Usecases:

This module can be used for spam mails detection in some website.

This module can be used by the sensitive departments of a country to see, who is making something wrong over the social media.

By converting a social media video into the text, we can analyse that what kind of content is present in the entire video.

We can use this module for checking the palagirism.

There are a lot of other usecases are present also.


Specs:

The used language is JavaScript.

UI developed is simply build with HTML.

BootStrap 4 is used to cascade the UI.
